# Strudel Project Brief

Strudel is a Pi extension that solves the primitive overload problem by treating agent capabilities like ingredients in a kitchen—instead of cramming all skills, tools, MCPs, and subagents into the context window, it maintains a searchable Pantry index and lets agents compose recipes by intent, pulling only what each task needs while keeping the context window small and the agent's reach large. The core metaphor is culinary: just as you don't haul the entire grocery store into the kitchen to make a sandwich, Strudel lets agents search by intent, compose across different primitive kinds (skills + tools + MCPs), and execute curated recipes rather than defaulting to the same familiar handful of capabilities. Next on the roadmap is completing v1—the foundation layer that establishes the Pantry index, search-to-bake gateway, and skills-first proof-of-concept—before advancing to v2 which will add L2 generative intelligence (a "cell phone" for categorization and recipe assembly) and deterministic recipe execution with proper tool orchestration.
